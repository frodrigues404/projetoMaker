React é uma biblioteca da linguagem JavaScript utilizada para construir interfaces de usuário. Essa tecnologia foi desenvolvida pelo Facebook, em 2013, sendo um framework de codigo aberto.

Aplicações SPA

A sigla SPA vem de Single Page Applications, ou Aplicações de Página Única. Nesse modelo, não há a necessidade de se fazer requisições para carregamento de novas páginas. A aplicação seria “carregada” por inteiro na primeira requisição, onde todo o HTML, CSS e JavaScript necessários seriam carregados de uma vez.
